# EduSmart AI

EduSmart AI is an AI-powered personalized learning platform designed to improve student learning through adaptive course recommendations, AI tutoring, assessments, progress tracking, and teacher-support tools.

The project is based on an academic assignment document that describes EduSmart AI as a solution for personalized education, student performance monitoring, teacher course management, certification, and subscription handling. :contentReference[oaicite:0]{index=0}

---

## 📌 Problem Statement

Many students struggle with fixed learning methods, lack of personalized support, delayed feedback, and limited progress tracking. Teachers also face challenges in managing lessons, quizzes, grading, and student performance manually.

EduSmart AI aims to solve these problems by using AI to deliver personalized learning paths, automate assessments, support teachers, and provide useful insights for students, parents, and institutions.

---

## 🎯 Objectives

- Provide personalized learning paths for students
- Recommend courses based on goals and performance
- Offer AI tutoring and doubt resolution
- Generate quizzes, assignments, and assessments
- Track student progress and learning gaps
- Help teachers manage lessons, courses, and grading
- Support certification and credential generation
- Manage subscriptions, payments, and platform users

## 🧩 Major Use Cases

1. **Student Registration Case**  
   A new student creates an account, completes profile setup, and receives platform access.

2. **Learning Path and Course Recommendation Case**  
   The platform recommends learning paths and courses based on student goals, skill level, and performance.

3. **Quiz and Assessment Case**  
   Teachers or AI generate quizzes, assessments, and diagnostic tests.

4. **AI Tutoring and Doubt Resolution Case**  
   Students ask questions and receive AI-generated explanations or teacher-supported responses.

5. **Student Progress Tracking Case**  
   The system tracks attendance, scores, progress, course completion, and learning gaps.

6. **Teacher Course Management Case**  
   Teachers create lessons, assignments, quizzes, and course materials.

7. **Certification and Credential Case**  
   Students receive certificates, badges, or credentials after completing learning milestones.

8. **Subscription and Payment Case**  
   The platform manages subscriptions, renewals, upgrades, cancellations, and payments.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/edusmart-ai.git
```

Navigate to the project folder:

```bash
cd edusmart-ai
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```bash
cp .env.example .env
```

Start the development server:

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add the following:

```env
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_jwt_secret
AI_API_KEY=your_ai_api_key
PAYMENT_SECRET_KEY=your_payment_secret_key
```
